# 6.2 Stetigkeit

Auch bei Funktionen von mehreren unabhängigen Variablen muss die Stetigkeit
definiert werden.

## Lernziele

```{admonition} Lernziele
:class: hint
Sie wissen, wann eine Funktion von mehreren unabhängigen Variablen **stetig** genannt wird.
```
